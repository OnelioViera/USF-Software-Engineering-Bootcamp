JavaScript Fundamentals

Fuctions

A JavaScript function is a reusable block of code that performs a specific task or set of tasks. It is defined using the "function" keyword followed by a name (optional) and a set of parentheses that may contain parameters. The code inside the function is executed when it is called, and it can return a value or modify the program state. Here is an example of a simple JavaScript function:

function greet(name) {
console.log("Hello, " + name + "!");
}

greet("Alice"); // Output: "Hello, Alice!"
greet("Bob"); // Output: "Hello, Bob!"
