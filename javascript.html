4.1 Expectation Setting for JavaScript

In this unit, you’ll get to the real meat of the course: JavaScript. You’ll primarily use JavaScript for the rest of the work that you do in both portions of this course.

So JavaScript is one of the most versatile and useful programming languages. There are endless options for what you can do with it. We'll talk more about exactly what it is and what it enables you to do in the next video.

For now, we want to set some context and expectations around JavaScript. JavaScript is going to take a lot longer to learn relative to things like HTML and CSS. It is also going to be more challenging. That's completely normal.

In this unit, you'll be learning all the foundational skills you need to jumpstart your JavaScript development journey. These concepts might be a little tricky at first because HTML and CSS are visual. You do something, you change the CSS in a file, you save it, you can see that change immediately in the browser. Same thing with HTML. Some of the things you'll use JavaScript are visual, and you’ll learn more about that in the Core Curriculum. The JavaScript that you're learning right now in Foundations is not going to be immediately visual.

It's totally normal if struggle a bit with JavaScript. You’ll be learning a lot of fundamental programming concepts in this unit which will be unfamiliar to you and the way that you approach problem-solving. These are concepts like variables and loops, conditionals, logic, data structures, functions. Understanding these ideas will give you a foundation for a way of structuring your thoughts and solving problems.

Programming concepts are bigger than just JavaScript. Understanding them will allow you to pick up another language like Python or C# much more easily. But learning software engineering thinking will take time. This JavaScript unit may take longer than HTML and CSS, that is totally normal. Gaining these basic JavaScript skills is what is going to set you up for success as a developer and the Core curriculum.

At the end of this unit, you're going to be ready to take the technical skills survey, the TSS. I know it's a bit intimidating. All that you really need to know right now is that it focuses on JavaScript. It's going to test you on your JavaScript abilities and your coding fundamentals. We don't care about whether you remember the exact syntax for JavaScript perfectly on your first go around, we want to make sure that you are capable of thinking like a programmer and that takes time so you may not get it immediately.

Lastly, you may notice that the videos in this unit do look different from previous videos. They have were taken from another course that Colt built. They've been edited and tweaked to fit in this course, but just know that's why they look a little bit different.
